Students' Performance analysis

For this project, I analyzed student performance data using Excel. To make sure everything was accurate and consistent, I started by cleaning the data with Power Query. After that, I organized important variables and relationships using data mapping. In order to analyze the data, I used the CORREL function to look for correlations between variables such as GPA and absences, as well as descriptive statistics to summarize the data. In order to make the data easier to understand and to support my recommendations, I finally produced visuals to highlight important insights and patterns..
